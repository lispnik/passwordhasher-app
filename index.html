This is basic port of http://wijjo.com/passhash/, a Firefox extension for computing a complex password from a site name and a passphrase. It is meant to be an experiment in building a HTML5 application for Android using AngularJS within an Android WebView component.

It actually turned out to be useful as well.

This project is maintained by lispnik
